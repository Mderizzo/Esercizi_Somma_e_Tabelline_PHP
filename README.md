# Esercizio Somma 

Il primo esercizio riguarda il calcolo della somma di un array di numeri: viene definito un array, inizializzata una variabile di somma e, attraverso un ciclo `foreach`, ogni elemento viene aggiunto alla somma totale che viene poi stampata a schermo. 

# Esercizio Somma

Il secondo esercizio riguarda invece la generazione della tabellina di un numero scelto dall’utente. Tramite un form HTML l’utente inserisce un valore numerico e, una volta inviato, il backend PHP lo elabora producendo la tabellina da zero a dieci, stampando i risultati riga per riga. Questi esempi mostrano l’uso combinato di HTML e PHP per gestire input e output, dimostrando come sia possibile integrare logica di calcolo e interfaccia utente in modo semplice ed efficace.
